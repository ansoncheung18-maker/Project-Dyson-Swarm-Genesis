## 2. 問題：人類嘅能源極限

| 指標 | 數值 |
|:---|:---|
| 目前全球能源消耗 | ~2.0 × 10¹³ W（20,000 GW） |
| 2100 年預計需求 | ~6.0 × 10¹³ W（目前嘅 3 倍） |
| 地球可接收太陽能（表面） | ~1.7 × 10¹⁷ W |
| 地球可實際使用太陽能（考慮損耗後） | ~1.0 × 10¹⁵ W |
| **地球-based 太陽能嘅實際極限** | **約目前用量嘅 50 倍** |

**限制**：即使我哋用晒地球所有陸地嚟鋪太陽能板，都會達到物理極限。要突破，就必須**喺太空收集能源**。

---

## 3. 解決方案：戴森蜂群

戴森蜂群係一個由數百萬個太陽能收集衛星組成嘅星座，圍繞太陽運行。同實心戴森球唔同，蜂群具有：
- **模塊化** — 可以逐步建造
- **可維護** — 每個衛星都可以維修或更換
- **經濟上可行** — 由小規模開始，幾十年內逐步擴展

### 3.1 點解揀 0.2%？

| 參數 | 數值 |
|:---|:---|
| 太陽總輸出功率 | 3.846 × 10²⁶ W |
| 0.2% 太陽輸出 | 7.69 × 10²³ W |
| 傳輸同轉換損耗 | ~70%（雷射 → 接收器 → 電網） |
| **實際可用能量** | **~2.3 × 10²³ W** |
| ÷ 人類目前用量（2.0 × 10¹³ W） | **~11,500 倍（≈12,000 倍）** |

**點解 0.2% 係可行嘅：**
- 喺 0.2 AU 軌道需要嘅收集面積：~1.0 × 10¹² m²（≈100 萬 km²）
- 呢個面積細過埃及嘅國土面積（100 萬 km²）
- 用 25 年時間，配合穀神星 ISRU 同機械人複製技術，係現實可行嘅

### 3.2 核心技術突破

| 技術 | 說明 |
|:---|:---|
| **穀神星一站式開採** | 金屬、矽、碳全部喺同一個天體開採 |
| **機械人自我複製** | 5 款機械人（R1-R5）可以自行複製，指數增長 |
| **指揮鏈架構** | 雲端大腦（AI）→ 區域指揮（FPGA）→ 中繼站（MCU）→ 無晶片機械人 |
| **能量增殖（Q=3）** | 核聚變能量增益驅動整個複製過程 |

## 4. 技術架構

### 4.1 系統方塊圖
穀神星（Ceres）
│ 開採金屬、矽、碳
▼
中央加工廠（穀神星）
│ 製成標準零件
▼
機械人複製（R1-R5）
│ 指數增長（22年內達到 3.02 × 10¹⁴ 個）
▼
衛星製造（R4 + R5）
│ 製造 7.54 × 10¹³ 個衛星
▼
衛星部署（0.2-0.3 AU 軌道）
│ 雷射傳輸
▼
地球接收站
│ 提供人類 12,000 倍能量

### 4.2 衛星規格

| 參數 | 數值 | 來源 / 理由 |
|:---|:---|:---|
| **收集面積** | 1 km² | VCCE/MEEP 基準 |
| **太陽輻射強度** | 35 kW/m² | 喺 0.2 AU（地球係 1.36 kW/m²） |
| **太陽能電池效率** | 40% | 高溫 GaAs（MEEP） |
| **毛輸出功率** | 14 GW | 1,000,000 m² × 35,000 W/m² × 0.40 |
| **淨輸出功率** | 10.2 GW | 扣除冷卻、轉換、傳輸損耗 |
| **冷卻系統** | 液態鈉 | 主動冷卻，保持 <150°C |
| **每個衛星質量** | ~1,500 噸 | |
| **推進系統** | 離子推進器 | 軌道維持同調整 |
| **設計壽命** | 30 年 | 可更換模塊 |

### 4.3 雷射傳輸系統

| 參數 | 數值 |
|:---|:---|
| 波長 | 1064 nm |
| 效率（DC → 雷射） | 90% |
| 自適應光學 | 有（Luna-Grid 技術） |
| 傳輸效率 | 85%（地球接收站） |
| 光束到達地球時嘅直徑 | ~100 km（安全、低強度） |
| 接收器技術 | 光伏整流天線（GaAs） |
