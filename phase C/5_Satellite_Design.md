## 1. 衛星總覽

戴森蜂群衛星係整個計劃嘅核心。每個衛星都係一個獨立嘅太陽能收集單元，負責收集太陽能並以雷射形式傳輸到地球。

| 參數 | 數值 | 備註 |
|:---|:---|:---|
| 衛星總數 | **7.54 × 10¹³ 個** | 0.2% 太陽能量目標 |
| 每個衛星質量 | **1,500 噸** | |
| 每個衛星功率（淨） | **10.2 GW** | 到達地球後 |
| 軌道半徑 | **0.2 – 0.3 AU** | 三個殼層 |
| 設計壽命 | **30 年** | 可更換模塊 |
| 材料來源 | **100% 穀神星 ISRU** | 金屬 + 矽 + 碳 |
| 控制方式 | **指揮鏈（無晶片）** | 被動穩定 + 遠程控制 |

## 3. 太陽能收集模塊

### 3.1 設計參數

| 參數 | 數值 | 備註 |
|:---|:---|:---|
| 收集面積 | **1 km²** | 有效收集面積 |
| 結構設計 | **3D 垂直光柵** | Luna-Grid 技術 |
| 太陽能電池 | **高溫 GaAs/Perovskite** | 效率 40%，MEEP 技術 |
| 工作溫度 | **<150°C** | 液態鈉冷卻 |
| 支架材料 | **金屬（穀神星）** | 鋁合金 |

### 3.2 3D 垂直光柵設計

| 參數 | 數值 |
|:---|:---|
| 光柵高度 | **50m** |
| 光柵間距 | **10m** |
| 塔陣列 | **200 × 200 塔** |
| 總收集面積 | **1 km²** |

**設計原理**：
- 傳統平面太陽能板只能捕捉直射陽光
- 3D 垂直光柵可以捕捉多角度陽光（直射 + 反射 + 散射）
- 有效收集面積提升 **5 倍**

### 3.3 太陽能電池規格

| 參數 | 數值 |
|:---|:---|
| 電池類型 | **GaAs/Perovskite 疊層** |
| 效率 | **40%** |
| 耐溫範圍 | **-200°C 至 +450°C** |
| 抗輻射 | **抗輻射設計** |
| 材料來源 | **穀神星矽（經 ISRU 提純）** |


## 4. 電力調節模塊

### 4.1 設計參數

| 參數 | 數值 | 備註 |
|:---|:---|:---|
| 輸入功率 | **14 GW** | 太陽能板毛輸出 |
| 輸出功率 | **13.3 GW** | 扣除冷卻耗電 |
| 轉換效率 | **> 95%** | DC-DC 轉換 |
| 電壓等級 | **100 kV DC** | 高壓傳輸 |
| 冗餘設計 | **3 個獨立通道** | N+1 冗餘 |

### 4.2 功率流向

太陽能板（14 GW）
│
├── 冷卻系統（5%）→ 0.7 GW
│
└── 剩餘 13.3 GW
│
└── 雷射轉換（90% 效率）
│
└── 雷射輸出 12.0 GW
│
└── 傳輸損耗（太空 → 地球 85% 效率）
│
└── 地球接收 10.2 GW
