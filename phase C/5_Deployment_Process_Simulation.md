# Phase C：衛星部署過程模擬
**專案**：戴森蜂群起源 (Dyson Swarm Genesis)  
**版本**：1.0  
**日期**：2026-06-30  
**作者**：Anson Cheung（14歲）


## 1. 模擬目標

模擬戴森蜂群衛星由穀神星製造、出發、飄移到 0.25 AU 戴森軌道嘅完整時間線，驗證 25-30 年內完成部署嘅可行性。


## 2. 關鍵假設

| 參數 | 數值 | 備註 |
|:---|:---|:---|
| 衛星總數 | 7.54 × 10¹³ 個 | 0.2% 太陽能量目標 |
| 每個衛星質量 | 1,500 噸 | |
| 太陽帆面積 | 1 km² | 衛星背面 |
| 激光輔助初始速度 | 1 km/s | 由穀神星激光陣列提供 |
| 到達時間（1 km/s 初始速度） | ~10 年 | 數值模擬結果 |
| 衛星製造速度 | 0.625 日/個 | 10,000 個 R4 機械人同時工作 |
| 機械人複製時間 | 22 年 | 由 Phase B 模擬 |
| 能量增益（Q=3） | 持續 | 50 座核聚變電廠 |


## 3. 模擬結果（核心數據）

### 3.1 衛星製造速度

| 參數 | 數值 |
|:---|:---|
| 每個衛星組裝時間 | 15 小時（0.625 日） |
| R4 機械人數量 | 10,000 個 |
| 每日可製造衛星 | 10,000 ÷ 0.625 = **16,000 個/日** |
| 每年可製造衛星 | 16,000 × 365 = **5.84 × 10⁶ 個/年** |

### 3.2 衛星飄移時間（太陽帆 + 1 km/s 初始速度）

| 初始速度 | 到達 0.25 AU 時間 | 相比純太陽帆改善 |
|:---|:---|:---|
| 0 km/s（純太陽帆） | 22.72 年 | 基準 |
| **1 km/s** | **~10 年** | **快 12.7 年** |
| 2 km/s | ~6 年 | 快 16.7 年 |

### 3.3 能量需求

| 項目 | 數值 |
|:---|:---|
| 每個衛星需要激光能量（加速到 1 km/s） | 208 MWh |
| 總能量需求（所有衛星） | 1.57 × 10¹⁶ MWh |
| 50 座核聚變電廠 + Q=3 能量增益 | **19 年累積足夠** |
